Qué se desarrolló y dónde estamos

✓ Funciona en plan Basic

  • ✓ Builder visual de páginas (bento-grid) integrado al editor de Shopify
  • ✓ Bloques de contenido: texto, imagen, botón, slider, carrusel
  • ✓ Tarjetas de producto con formato mayorista
  • ✓ Spinner de cantidad y compra rápida sin recarga
  • ✓ Metafields de producto (bulto, contenido, tiers)
  • ✓ Contact gate: registro + sincronización con Google Sheets
  • ✓ Visualización de descuentos por tier (solo visual)

✗ No funciona en plan Basic

  • ✗ Aplicación real de descuentos en checkout — Los descuentos se muestran visualmente pero no se aplican al precio final

El problema: La app de descuentos usa Shopify Functions, una tecnología que requiere mínimo el plan "Shopify" ($105/mo). En el plan Basic, la función nunca se ejecuta y el cliente paga precio completo.

Nuestra recomendación

Opción A — Upgrade al plan Shopify

Es la solución más eficiente en relación costo-beneficio para el corto plazo:

  • $ +$66 USD/mes es un costo menor comparado con el desarrollo ya invertido y el valor que genera la lógica mayorista completa
  • ⚡ Cero tiempo de implementación — todo funciona inmediatamente al cambiar el plan
  • ✓ Se aprovecha el 100% del desarrollo ya realizado: builder visual, lógica de descuentos, contact gate y app
  • ⚙ Incluye Shopify Flow para futuras automatizaciones (emails, etiquetas, notificaciones)

★ A mediano plazo

Si el objetivo es reducir costos recurrentes y tener mayor control, WooCommerce es la mejor alternativa: equilibrio entre funcionalidad, costo y autonomía. Desarrollo a medida solo se justifica si hay necesidades muy específicas o integración con sistemas internos.

✗ No recomendadas

  • Códigos auto-aplicados — un solo descuento por orden es incompatible con tiers diferentes por producto
  • Draft Orders — el costo de desarrollo y mantenimiento supera los $66/mes del upgrade
